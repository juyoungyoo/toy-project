# Small shopping mall
> Spring boot & JPA 활용 infrean 강의 실습을 하며, JPA study 합니다.


#### Spec 
- REST API
- Spring boot 2.x
- JPA 
- Gradle 
- Lombok
- H2  
- Thymeleaf

#### DB modeling
- 회원:주문 = 1:N
- 주문:상품 = N:N


#### 요구사항
- [ ] 회원은 상품을 주문할 수 있다.
- [ ] 주문 시 여러 종류의 상품을 선택할 수 있다. 

__[회원기능]__
- [ ] 회원 등록
- [ ] 회원 조회
        
__[상품기능]__ 
- [ ] 상품 등록
- [ ] 상품 수정
- [ ] 상품 조회

__[주문기능]__

- [ ] 주문 내역 조회
